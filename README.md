# snake_game

A simple console‑based Snake game implemented in C++.  
This project was created to practice C++ fundamentals, game loops, input handling, and basic state management in a terminal environment.

---

## Features

- Classic Snake gameplay in the console
- Real‑time movement using keyboard input
- Food spawning and score tracking
- Game‑over detection (self‑collision or wall collision)
- Single‑file implementation for clarity and learning


---

## Build

Compile with g++:

```sh
g++ main.cpp -o snake
